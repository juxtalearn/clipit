\ElggMetadata

ElggMetadata This class describes metadata that can be attached to ElggEntities.

Extenders allow you to attach extended information to an ElggEntity. Core supports two: ElggAnnotation and ElggMetadata.

Saving the extender data to database is handled by the child class.

Compatibility for subclassing in 1.7 -> 1.8 change. Passing true (default) emits a deprecation notice. Passing false returns false. Core constructors always pass false. Does nothing either way since attributes are initialized by the time this is called.
