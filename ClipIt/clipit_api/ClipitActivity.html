Class ClipitActivity

Represents a JuxtaLearn cycle of tasks around a selected Tricky Tipic. It contains users (which have been called to join the activity), groups (that actually participate in the activity), tasks (set by the teacher/s), videos and files.

An Activity can have different status: enroll state, active state and closed state.
